**#📚 AI Powered Study Buddy**



**## Overview**



AI Powered Study Buddy is an AI-based learning assistant built using Python, Streamlit, Google Gemini API.



It allows students to upload PDF notes and perform AI- powered learning tasks.



**## Features**



* Upload PDF Notes
* Generate Summary
* Ask Questions
* Generate Flashcards
* Generate Quiz
* Download Outputs





**## Technologies Used**



* Python Streamlit 
* Google Gemini API
* PyPDF
* python-dotenv





**## Installed**



```bash

pip install -r requirements.txt

streamlit run app.py

```
